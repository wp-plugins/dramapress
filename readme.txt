== Description ==

This is not just a plugin, it symbolizes the angst and ire of an entire generation summed up in a few words. When activated you will randomly see WordPress related drama spewing forth. More drama will be added as it becomes available.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `dramapress.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
